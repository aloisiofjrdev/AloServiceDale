## Installation

AloServiceModel is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'AloServiceModel',
:git => 'https://github.com/aloisiofjrdev/AloServiceDale.git',
:branch => 'master'

```
